# 统计术语

  1. 同比：与历史同期相比，往往指去年同期
  2. 环比：与相邻的上一个统计周期相比，一般看末尾的时间（即精确到哪个时间段）
  3. 基期：作为对比参考的时期
  4. 现期：相对于基期的时期
  5. 增长量：增长量=现期量-基期量=基期量×增长率=$\frac{现期量}{1+增长率}\times 增长率$
  6. 增长率：增长率=$\frac{增长量}{基期量}$=$\frac{增长量}{现期量-增长量}$=$\frac{现期量-基期量}{基期量}$
  7. 变化幅度/涨跌幅度：比较绝对值
  8. 比重：比重=$\frac{部分量}{整体量}$
  9. 倍数：倍数=$\frac{A}{B}$,增长了多少倍=$\frac{A}{B}-1$=$\frac{A-B}{B}$
  10. 年均增长量：年均增长量=$\frac{末期量-初期量}{相差年数}$
  11. 年均增长率：$末期量=初期量\times (1+年均增长率)^n$，n为相差年数
  12. 发展速度：发展速度=$\frac{现期量}{基期量}$=1+增长率
  13. 拉动增长率：拉动增长率=$\frac{部分增长量}{整体基期量}$
  14. 增长贡献率：增长贡献率=$\frac{部分增长量}{整体增长量}$
  15. 成：一成等于十分之一
  16. 番：翻n番即为原来的$2^n$倍
  17. 顺差：在一个时期类，一个国家或地区的出口商品额大于进口商品额，顺差额=出口额-进口额
  18. 逆差：在一个时期类，一个国家或地区的出口商品额小于进口商品额，逆差额=进口额-出口额
  19. 国内生产总值
    1. 国内生产总值由第一产业、第二产业、第三产业增加值构成
    2. 三大产业的产值经常称为产业增加值，而不是相对增长量
  20. 十三五：2016-2020
  21. 十四五：2021-2025

# 速算技巧

  1. 加减法
    1. 尾数法：选项与材料精确度一致且尾数不同
    2. 截位法：选项与材料精确度不一致且和差粗略计算
    3. 精确计算：选项非常接近（前两位或三位相同）
  2. 乘除法
    1. 左二截：选项首位不同，将分母从左至右截取两位
    2. 留三位：选项首位相同，第二位不同，将分母从左向右截取三位
    3. 精确计算：选项首两位相同或很接近
    4. 截位舍相同：分子或分母出现加减法计算式，依据选项讲分子或分母进行截位，舍去相同的数位
  3. 特殊分数
    1. 常用于根据现期量和增长率计算增长量，将增长率转化为$\frac{1}{n}$
    2. 增长量n+1原则：$增长量=\frac{现期量\times 增长率}{1+增长率} = \frac{现期量\times \frac{1}{n}}{1+\frac{1}{n}}=\frac{现期量}{n+1}$
    3. 若$m\times n=100$,则$m%=\frac{1}{n}, n%=\frac{1}{m}$

# 重点题型

# 增长量

  1. 增长量计算
    1. 关键词：比去年/同比增长/减少了
    2. 公式：增长量=现期量-基期量=基期量×增长率=$\frac{现期量}{1+增长率}\times 增长率=\frac{现期量}{n+1}$
  2. 年均增长量计算
    1. 关键词：年均，平均每年
    2. 公式：年均增长量=$\frac{末期量-初期量}{相差年数}$
  3. 增长量比较
    1. 已知现期量和基期量，直接作差比较
    2. 已知现期量和增长率：大大则大（即现期和增长率均大则大），一大一小看乘积（即直接使用公式计算较为麻烦，直接使用现期量乘增长率进行估计比较）

# 增长率

  1. 普通增长率
    1. 关键词：比去年，同比，环比增长了%，增速为
    2. 公式：增长率=$\frac{增长量}{基期量}$=$\frac{增长量}{现期量-增长量}$=$\frac{现期量-基期量}{基期量}=\frac{现期量}{基期量}-1$
  2. 普通增长率比较
    1. 增长率（增速）的差值单位为“个百分点” 而不是%
    2. 若题干给出增长率则直接读数比较
    3. 若题干给出基期量与现期量
      1. 基期量差别不大，直接比较增占良
      2. 使用$\frac{现期量}{基期量}$
  3. 间隔增长率
    1. 关键词：现期与基期间隔一期
    2. 公式：$R=r_1+r_2+r_1\times r_2$
    3. 若$r_1r_2<25$,则$R=r_1+r_2+r_1\times r_2\approx 1+r_1+r_2$，常用于近似计算
    4. 若间隔多个周期，且只判断是否增长时，有以下规律
      1. 对于多个周期间增长率依次为：a,b,c,d,e...，判断总增长率与1的关系
      2. 若a+b+c+...<0,则(1+a)(1+b)(1+c)...<1,当且仅当a,b,c,d...均为0时可取等号
  4. 混合增长率
    1. 关键词：两个或多个部分合成一个整体，并求整体或其中一部分的增长率
    2. 比较思路：混合增长率介于各部分增长率之间，偏向于基期量较大的部分所对应的增长率。即若整体增长率为m，部分增长率分别为a，b，且a的基期量大于b，并且a<b, 则a<m<b，且m-a<b-m
  5. 年均增长率
    1. 关键词：年均，平均每年增长，年均增速
    2. 比较式，使用$\frac{现期量}{基期量}$
    3. 计算时，由于开方运算较为复杂，一般使用代入排除法

# 比重

  1. 现期比重：比重=$\frac{部分量}{整体量}$
  2. 基期比重：$\frac{A}{B}\times \frac{1+b}{1+a}，其中A,a为部分，B，b为整体$
  3. 两期比重
    1. 比较：比重大小关系与a,b关系相同
    2. 差值：$\frac{A}{B}\times \frac{a-b}{1+a}，其中A,a为部分，B，b为整体$
  4. 十字交叉法：适用于已知部分平均值和整体平均值，求部分数量之比
    1. 若有两种质量分别为 A 与 B 的溶液，其浓度分别为 a 与 b，混合后浓度为 r，则由溶质质量不变可列出下式 Aa+Bb=(A+B)×r

# 综合分析

  1. 关键词：以下说法正确/错误的是，从资料中推出
  2. 思路：
    1. 原则：
      1. 信息易得：先看与材料时间一致的选项，减少计算
      2. 单个优先：单个计算优于多个比较，时间点优于时间段计算
      3. 难题不算：计算量特别大的可以先跳过
      4. 易错跳过：成数（比重)、翻番、均、都、不足、逆顺差的选项一般是错误的
    2. 优先：
      1. 读数优先：可以直接读数的选项优先
      2. 简单计算优先：加减法运算的优先
      3. 口诀优先：增长量比较、混合增长率、两期比重比较等有口诀的优先
      4. 基础公式优先：增长率化除为乘、增长量、间隔增长率优先
