# 图形推理

# 位置类

  1. 动态位置：
    1. 平移：
      1. 题型特征：组成元素相同、元素个数种类相同
      2. 上下平移、左右平移、顺时针平移、对角线平移
    2. 旋转：
      1. 题型特征：组成元素相同、图形整体或部分方向不同
      2. 顺时针、逆时针
    3. 翻转：
      1. 题型特征：组成元素相同个，有图形呈现轴对称
      2. 上下翻转、左右翻转
  2. 静态位置
    1. 相离：内含、外离
    2. 相连：点链接、线连接
    3. 相交
    4. 相同元素排布
      1. 直线排布：相邻、相间
      2. 环形排布：相邻、相对

# 样式类

  1. 遍历：
    1. 题型多为九宫格或两段式
    2. 所有出现的元素都要经历一遍
  2. 运算
    1. 相加、相减
      1. 提醒多为九宫格或两段式
      2. 组成元素相似，第三个图形比前两个图形的线条或元素明显增减或减少
    2. 求同
      1. 去异求同：前两个图形去除不同部分，剩余部分相同，得到第三个图形
      2. 相邻求同：相邻图形之间含有相同元素，不要求大小位置相同
      3. 整体求同：每行或每列或所有图形中含有相同元素，不要求大小和位置相同
    3. 求异：组成元素相似，一般图形中出现的线条较多，前两个图形去除相同部分，剩余不同部分
  3. 定义叠加
    1. 图形轮廓相同，在相同位置上填充的内容不同
    2. 填充内容：主要是黑色白色，也可以是其他小元素
    3. 形式多为九宫格或两组式
    4. 由同一位置的前后运算关系得出结论

# 数量类

  1. 数字规律：等差、常数、周期、比例等
  2. 题型特征：组成元素不同、较为凌乱，或某些对象数量变化明显
  3. 数的对象
    1. 点：
      1. 交点：顶点、切点、曲直交点、端点（较少考察）等
      2. 黑白点
    2. 线
      1. 直线数、曲线数、总线数
      2. 笔画数：
        1. 奇点：引发出奇数条线条数的点。
        2. 一个连通图形奇点数为0或2，即为一笔画，否则笔画数=奇点数 $\div$ 2。多个连通图分开考虑
        3. “田”字是两笔画的特征图形，若出现田的变形图，优先考虑两笔画图形
    3. 角（考察较少）
      1. 数总角数
      2. 数特定角：锐角、直角、钝角。当图形中有垂直线条，直角特征明显是优先考虑数直角
    4. 面
      1. 指线条围成的封闭白色空间
      2. 题型特征：图形封闭特征明显
      3. 直线面、曲线面、特性形状的面（通常为形状相同的面）
    5. 素
      1. 指组成图形的元素
      2. 题型特征：由一种或多种小元素组成或由多个部分构成
      3. 通常数个数、种类数、部分数

# 属性类

  1. 对称性
    1. 轴对称图形
      1. 对称轴个数
      2. 对称轴方向
      3. 对称轴穿过点还是线，穿过的个数
    2. 中心对称图形
  2. 曲直性：曲线图形、直线图形、直曲图形
  3. 开闭性：开放图形，封闭图形（全封闭、半封闭）

# 功能类

  1. 题型特征：每幅图均含有明显突兀的小元素
  2. 功能点：
    1. 标定方位：上下左右、里外
    2. 标定空间：大空间、小空间、同一空间、公共空间等
    3. 标定特征：直线、曲线、长边、短边、角、线条、交点
  3. 功能箭头
    1. 指示方向：同向、异向，向里、向外
    2. 指示元素：指向小图形或元素
    3. 指示关系：两箭头平行、垂直
  4. 功能线
    1. 指示方向：横、竖、斜
    2. 指示关系：平行、垂直
    3. 连接元素：两端元素相同或不同

# 六面体

  1. 思路：观察面与面之间的位置关系，再根据选项使用排除法
  2. 相对面法
    1. 立体图形：上下、左右、前后互为相对面
    2. 用法
      1. 两个相对面在同一个立体图形中不可能同出现
      2. 两个相对面在立体图形中必出现一个
    3. 展开图
      1. 相间排列：同一行或同一列每间隔一个互为相对面
      2. Z字两端：中间只能隔一行或一列的Z字两端的面互为相对面
  3. 相邻面法
    1. 指在立体图形中存在公共点或公共边的多个面
    2. 相邻面的位置关在在折叠前后要相符，不能有变化
    3. 判断方法
      1. 移面法：同一行两端（只有一条相邻边的，如图6中的1，5）可进行平移，有90度的边可进行滚动
      2. 箭头法：借用箭头来确定平面展开图与立体图两个面或者三个面的相邻关系
      3. 公共棱/公共点法：通过画出公共边或点的方式比较展开图与立体图两个相邻面相邻关系是否一致

# 立体拼合

  1. 题型特征：寻找立体图的所缺部分
  2. 原理：凹凸对应
  3. 思路：对比凹凸的形状、位置、数量
  4. 注意：有时需进行一定的旋转

# 视图类

  1. 题型特征：根据立体图找视图
  2. 思路：观察选项-对比题干
  3. 要点：可见轮廓用实现，不可见轮廓用虚线或不画

# 截面图

  1. 给出立体图找出对应界面
  2. 思路：排除法

# 定义判断

  1. 题型特征：题干给出一个或多个定义，选出最符合或最不符合定义的选项
  2. 思路：看清设问-找到定义中的重点-对比选项
  3. 单定义判断
    1. 主体：行为或动作的发起者
    2. 客体：行为或动作的承受者
    3. 定语：表修饰、限定、说明的名词，主动被动、有益有害、积极消极、相同不同、主观客观、暂时永久、直接间接
    4. 状语：修饰时间、地点、原因、目的、方式、结果
  4. 多定义判断
    1. 类别：并列型、包含型
    2. 思路
      1. 并列型：只看所问的定义
      2. 包含型：看所问定义和主定义，找出题设中的指定从定义
      3. 题设未明确：全部都看

# 类比推断

# 外延关系

  1. 全同关系：两个概念表达的是同一事物
  2. 并列关系
    1. 矛盾关系：同一概念下非此即彼
    2. 反对关系：同一概念线互不相容，但除了这两个还有别的选择
  3. 交叉关系：两个概念间只有一部分重合的关系，即$A\neq B, A\cap B \neq \emptyset$
  4. 包容关系
    1. 种属关系：A是一种B
    2. 组成关系：A是B的组成成分
  5. 全异关系：两个概念没有任何重合

# 内涵关系

  1. 属性关系
    1. 必然属性：某一类事物必然具有的某种属性
    2. 或然属性：某一类事物可能具有的某种属性
  2. 条件关系
    1. 充分条件：如果有A，一定有B，如果没有A，不一定没有B
    2. 必要条件：如果没有A，一定没有B，如果有A,不一定有B
  3. 对应关系：功能、地点、目的、因果、时间顺承、成品与原材料、配套使用

# 语义关系

  1. 近义、翻译、比喻象征

# 语法关系

  1. 主谓、主宾、谓宾、偏正（形容词修饰名词，副词修饰动词）

# 逻辑关系

# 翻译推理

  1. 题型特征：含有明显的逻辑关联词，如果那么、只有才、且、火车
  2. 思路：先翻译（将有逻辑关联词的句子翻译为逻辑表达式），再根据规则推理（）
  3. 翻译规则
    1. 如果那么，前推后：如果A，那么B。翻译为A->B
    2. 只有才，后推前：只有A，才能B。翻译为B->A
  4. 推理规则：肯前必肯后，否后必否前，否前肯后不必然。即翻译结果为A->B，则A->B, $\neg B\rightarrow\neg A$为确定项。B->A, $\neg A\rightarrow\neg B$为非必然情况。
  5. 且关系、或关系：$\neg(A\land B) = \neg A \lor \neg B, \neg(A\lor B) = \neg A \land \neg B$

# 分析推理

  1. 题型特征：题干给出一组对象级相关的若干信息，要求对对象与信息进行匹配或排序
  2. 排除法：适用于题干所给信息确定且充分
  3. 最大信息法：利用于题干出现频率最高、数量最多的有效信息作为推理起点。适用于题目无法直接利用排除法，且存在高频有效信息
  4. 确定信息法：以提噶种给出的确定信息作为推理起点。适用于题目中有能够确定真假的信息，且无法直接利用排除法推出
  5. 代入法：题干给出的条件有真有假，且无法找出确定信息，选项不容易排除

# 归纳推理

  1. 题型特征：题干与选项均无明显逻辑关联词，设问为由此可以/不能推出
  2. 解题原则
    1. 话题一致
    2. 现在原则：能够推出的选项叙述的时间只局限于题干内容
    3. 可能优先原则：可能性说法正确概率大于绝对性说法
    4. 敏感词汇慎选原则：表比较和趋势的选项慎选，首要、主要、更、最、越等

# 加强论证

  1. 题型特征：提问方式含支持、加强、前提、假设等
  2. 思路：确定题型-找论点-找论据-分析选项
  3. 加强方式
    1. 加强论点：重复论点
    2. 加强论据：肯定原有论据、增加新论据
    3. 建立联系：建立论据与论点间的联系
    4. 隐含前提：增加一个论证成立的必要条件
  4. 因果加强论证
    1. 排他因素：除改原因外没有其他原因导致此结果。如对比实验中强调外在条件一致
    2. 没A没B：没因没果，说明二者是因果关系。如对比实验中强调对照组未进行某一行为65导致的结果不同
  5. 强度大小：建立联系>补充前提>增加论据>重复论点

# 削弱论证

  1. 题型特征：提问方式中含有削弱、质疑、反驳等
  2. 削弱方式
    1. 削弱论点：否定论点
    2. 削弱论据：否定原有论据、增加反向论据
    3. 切断联系：切断论据与论点之间的联系。削弱力度大于削弱论据
    4. 否定前提：否定论证成立的必要调教
  3. 因果削弱方式
    1. 因果倒置：颠倒原因和结果发生的顺序。削弱力度大于另有他因
    2. 另有他因：除改原因外还有其他的原因导致结果。自身的另有他因削弱力度大于周围环境的另有他因
    3. 没A有B：没有改原因，但导致了这一结果
    4. 有A没B：有改原因，但没有产生这一结果

# 真假推理

  1. 题型特征：题干中有若干论断，提问方式有真假限定（即有xx个为真）
  2. 思路：先找关系，再看其余
  3. 矛盾关系：
    1. 必有一真，必有一假
    2. 常见矛盾关系
      1. $A与\neg A$
      2. $\forall A \in B 与 \exists A \notin B$
      3. $\forall A \notin B 与 \exists A \in B$
      4. A -> B与 $A \land \neg B$
      5. $A \land B 与 \neg A \lor \neg B$
      6. $A \lor B 与 \neg A \land \neg B$
  4. 反对关系
    1. 两个所有，必有一假，可以同假。如$\forall A \in B 与 \forall A \notin B$
    2. 两个有的，必有一真，可以同真。如$\exists A \in B 与 \exists A \notin B$

# 原因解释

  1. 题型特征：以下哪项如果为真，最能/不能解释题干中的现象/矛盾
  2. 解题技巧
    1. 找矛盾/现象：利用语义或转折词
    2. 找原因
      1. 若有矛盾，优先选择涉及矛盾双方的选项，次选涉及单方的选项
      2. 若无矛盾，只有普通现象，则排除无关选项后择优选择

# 平行结构

  1. 题型特征：下列哪个推理结构/论证方式与题干相同
  2. 推理结构分类
    1. 命题结构类：先利用翻译推理中的方法将题干翻译为形如A->B的形式，再翻译选项
    2. 论证方式类
      1. 演绎论证：从一般到特殊
      2. 归纳论证：从部分或全部推出一般。（如抽样调查/全体调查得出一般结论）
      3. 类比论证：从特殊到特殊。（如从动物实验类比到人）
    3. 其他
      1. 偷换概念：如从猫猜想熊猫
      2. 自相矛盾
      3. 答非所问
