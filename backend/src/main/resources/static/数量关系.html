# 常用解题方法

# 带入排除法

  1. 适用题型
    1. 选项信息充分：每个选项含两个或两个以上的数据
    2. 固定题型：不定方程、多位数问题（题目中涉及多个数位）、余数问题、年龄问题
  2. 思路
    1. 优先排除：排除不符合题意的选项
    2. 最值代入：问题涉及最大最小时，优先代入最大最小的数据
    3. 从简代入：非最大最小问题，优先带入方便计算的选项

# 数字特征法

  1. 奇偶特征
    1. 适用题型
      1. 包含求和、求差
      2. 不定方程问题、约数、倍数、余数、平均数问题
    2. 技巧
      1. 和差：同奇同偶为偶，一奇一偶为奇
      2. 和为奇，差也为奇，和为偶，差也为偶
      3. 乘积：至少一个乘数为偶，积为偶。乘数均为奇，积为奇
  2. 倍数特征
    1. 适用题型
      1. 题目中出现较多分数、百分数、比例、倍数、余数、平均数
      2. 题型：不定方程问题、约数倍数问题、余数问题、平均数问题
    2. 常见倍数判定法
      1. 是否为2/4/8的倍数，看能末尾1/2/3位数能否被2/4/8整除
      2. 是否为3/9的倍数，看各位数之和能否被3/9整除
      3. 若A、B互质，判断能否被AxB整除，只需看能否被A和B整除（判断能否被某个数整时，可将该数拆为两个互质的数相乘，分别判断）
      4. 若a:b = m:n，则a是m的倍数，b是n的倍数，(a±b)是(m±n)的倍数

# 方程法

  1. 方程（组）
    1. 适用题型：有等量关系词：总量、工、等于、比..多/少
    2. 常用公式：工作总量=效率x时间，路程=速度x时间，总价=单价x数量
    3. 步骤
      1. 设中间量或所求量，（中间量优先考虑比/是后的量）
      2. 根据等量关系列方程
      3. 解方程
  2. 不定方程（组）
    1. 适用题型：未知数个数大于方程个数
    2. 解方程方法：奇偶特性、倍数特性、尾数特性，结合代入法求解

# 枚举归纳法

  1. 适用题型：通过列举或归纳出某种规律的题目
  2. 思路：当满足条件的情况较少时适用枚举，当数目较多同时有明显规律时适用归纳

# 赋值法

  1. 适用题型：
    1. 题目没有给出具体量，如只包含比例、百分数
    2. 形如A=BxC的关系式最多只给出一个具体量
    3. 工程问题、形成问题、利润问题、浓度问题等
  2. 思路
    1. 对不变量进行赋值
    2. 赋值为给定数字的公倍数或比例关系
    3. 赋值尽量便于计算

# 数列规律问题

  1. 有线性趋势
    1. 增减幅相对较小，考虑作差。若三项以上无法找到规律，则转换思路
    2. 增减幅相对较大，考虑乘除
    3. 增减幅很大，且项位于幂次附近，考虑幂次。如2(1^1+1)，5(2^2+1)，28(3^3+1)，257(4^4+1)
  2. 无线性趋势
    1. 思路：寻找视觉冲击点--数列中存在着的相对特殊、与众不同的现象
    2. 长数列，项数在6项以上。基本解题思路是分组（如几个数一组求和）或隔项（如分隔奇数项和偶数项，各自呈规律）
    3. 摇摆数列，数值忽大忽小，呈摇摆状。基本解题思路是隔项。
    4. 题干出现两个括号。一定是隔项成规律
    5. 出现分式项，提示作乘除
    6. 正负交叠。基本思路是做商。
    7. 根式。将每一项放入根式下
    8. 首一项或首两项较小且接近，第二项或第三项突然数值变大。基本思路是分组递推，用首一项或首两项进行五则运算（包括乘方）得到下一个数。
    9. 纯小数数列，即数列各项都是小数。基本思路是将整数部分和小数部分分开考虑，或者各成单独的数列或者共同成规律。
    10. 很像连续自然数列而又不连贯的数列，考虑质数或合数列。（其和或差数列也有可能是质数或合数）
  3. 其他规律
    1. 约去公因数。
    2. 因式分解法。
    3. 通分法。

# 工程问题

  1. 基础公式型
    1. 公式：工作总量=工作效率x工作时间
    2. 特征：题干中有工作总量、效率、时间三个量中至少两个有具体数值
  2. 给定时间型
    1. 特征：题干只给出各主题完工需要时间
    2. 思路：赋值总量-计算各主题效率-求解
    3. 赋值技巧：一般赋值时间的公倍数为总量
  3. 效率制约型
    1. 特征：题干给出效率的比例倍数关系
    2. 思路：赋值效率-计算总量-求解
    3. 赋值技巧，按比例赋值效率，一般赋值为整数

# 经济利润问题

  1. 利润率折扣
    1. 基础公式
      1. 利润=售价-成本=成本x利润率
      2. 利润率=利润÷成本=(售价-成本)÷成本
      3. 售价=定价x折扣
      4. 折扣=售价÷定价
      5. 总价=单价x销量=未打折收入+打折后收入
      6. 总利润=总售价-总成本=单间利润x销量
  2. 分段计费
    1. 题型特征：题干出现明显分段区间，每个分段区间计费不同
    2. 思路：明确不同分段计费标准，分区段计算
  3. 统筹优化
    1. 题型特征：题干出现多种方案，每种方案计费方式不同，问题出现最多、最少得表述
    2. 思路：对比各种情况，常使用贪心思想

# 行程问题

  1. 基本行程问题
    1. 基础公式
      1. 路程=速度x时间
      2. 等距离平均速度 $\overline{v}=\frac{2v_1 \cdot v_2}{v_1 + v_2}$
      3. 火车过桥：
        1. 完全过桥：路程=桥长+车长=vt
        2. 车在桥上：路程=桥长-车长=vt
  2. 相遇追及问题
    1. 线型
      1. 相遇
        1. 路程=（大速度+小速度)x相遇时间
        2. 两端相向出发，到达一端立即折返，相遇n次：(2n-1)S=(大速度+小速度)x相遇时间
        3. 单端同向出发，到达一端立即折返，相遇n次：2nS=(大速度+小速度)x相遇时间
      2. 追及：追及距离=(大速度-小速度)x追及时间
    2. 环形
      1. 同一点相向出发，相遇n次：nS=(大速度+小速度)x相遇时间
      2. 同一点同向出发，追上n次：nS=(大速度-小速度)x相遇时间
  3. 流水行船问题
    1. 顺流：航程s=(船速+水流速)x顺流时间
    2. 逆流：航程s=(船速-水流速)x顺流时间

# 排列组合与概率问题

  1. 基础排列组合：分类加法，分布乘法，有序排列，无序组合
  2. 特殊排列组合
    1. 捆绑法：
      1. 关键词：相邻、相连、在一起，
      2. 思路：将相关元素看成一个元素，再进行排列或组合
    2. 插空法：
      1. 关键词：间隔、不相邻、不在一起，
      2. 思路：将可以相邻的元素排列，形成若干空位，将不相邻的元素插入空位
    3. 隔板法
      1. 关键词：n个相同的元素分为m个组，每个组至少1个
      2. 思路：$C_{n-1}^{m-1}$ ,可以理解为在n个元素形成的n-1个空中插入m-1个挡板
      3. 变形：若要求每个组至少分a个，可先给每个组分a-1个，再转换为至少一个的隔板法，$C_{n-m(a-1)}^{m-1}$
    4. 错位排列
      1. 例：n辆车停在n个车位，全部开出后再停回车位，但不能停到之前停的车位
      2. $D_n$表示n个元素错位排列，则满足$D_n=(D_{n-2} + D_{n-1}) \times (n-1)$, 其中$D_1 = 0, D_2 = 1$
      3. 常考1至5，$D_1 = 0, D_2 = 1, D_3 = 2, D_4 = 9, D_5 = 44$
  3. 概率问题
    1. 基础概率：P=满足条件数÷总情况数
    2. 分步用乘法，分类用加法
    3. 逆向思维：若正向情况较为复杂，则逆向思考，概率=1-不满足条件的概率

# 几何问题

  1. 公式：
    1. 球体表面积=$4\pi r^2$
  2. 几何性质
    1. 等比缩放：若将一个图形尺度扩大为原来的N倍，则
      1. 对应角度不变
      2. 对应周长扩大为N倍
      3. 面积扩大为$N^2$倍
      4. 体积扩大为$N^3$倍
    2. 最值
      1. 平面中，若周长一定，越接近于圆，面积越大
      2. 平面中，若面积一定，越接近于圆，周长越小
      3. 立体图形中，若表面积一定，越接近于球，体积越大
      4. 立体图形中，若体积一定，越接近于球，表面积越小

# 容斥原理

  1. 公式
    1. $A+B-A\cap B=总数-AB均不满足$
    2. $A+B+C-A\cap B - A\cap C - B\cap C + A\cap B\cap C = 总数-ABC均不满足$
    3. $总数-总数-ABC均不满足=A+B+C-只满足两个条件的-2\times 三个条件均满足的$

# 最值问题

  1. 最不利构造
    1. 关键词：至少...才能保证...
    2. 思路：
      1. 最不利情况数+1
      2. 保证值-1=最不利值
      3. 最不利=所有最不利值之和
      4. 分组讨论的情况，若某个情况数小于最不利值时，也要加到最不利值中
      5. 例：有a，b，c，d四组数，至少要选多少个才能保证有m个来自同一组，则若a>=m-1，则m-1加入答案，若b<m-1,则加入b
  2. 数列构造
    1. 关键词：最多(少)的...最多(少)为...，排名第N的最多(少)为
    2. 思路：设所求量为未知数-用贪心思想列方程-解方程
    3. 注意：
      1. 需考虑题目是否要求各数不相同
      2. 若非整数，至多需向下取整，至少向上取整
  3. 多集合反向构造
    1. 关键词，都...至少，...至少...都
    2. 思路：类似容斥原理和贪心思想
      1. 反向：分别求出各几何的反向数据
      2. 加和：将各反向数据直接相加，即认为各种反向情况没有交集，最大化反向的情况
      3. 作差，总数减去加和数据

# 时间问题

  1. 年龄问题
    1. 年龄=现在年份-出生年份
    2. 属相相同，年龄不同，则至少相差12岁。若为父子，则至少相差24岁
    3. 常使用代入法或方程法解题
  2. 星期日期问题
    1. 闰年：非世纪年能被4整除，世纪年能被400整除
    2. 周期推断：每隔n天为每(n+1)年，求最小公倍数
    3. 星期日期推断：连续7n天，周一至周日均出现n次
    4. 平年为52周余1，闰年余2，因此星期数过一平年加一，过一闰年加二

# 溶液问题

  1. 多次操作公式
    1. 倒出比例为a的溶液，再倒入相同的溶剂，浓度变为原来的(1-a)（即溶质倒出a）
    2. 加入比例为a的溶剂，再倒出相同的溶液，浓度变为原来的$\frac{1}{1+a}$（即加溶剂时改变浓度，倒出溶液时不改变浓度）

# 牛吃草问题

  1. 关键词：既有消耗，又有增长
  2. 公式：$y=(N-x)\times T$，y现有存量，N单位时间总消耗数，x单位时间增加数，T消耗完用时

# 比赛问题

  1. N支队伍循环赛
    1. 单循环：即与每支其他队伍进行一场比赛，总场次为$C_N^2$
    2. 双循环：即与每支其他队伍进行两场比赛，总场次$A_N^2$
  2. N支队伍进行淘汰赛：N支队伍两两进行比赛，输一场淘汰出局，每轮淘汰一半选手，直至产生冠军
    1. 决出冠亚军需要进行（N-1)场比赛，即每场比赛淘汰一支队伍
    2. 决出前4名，需要进行N场比赛，即多出一场三四名之间的比赛
    3. 每轮淘汰一半队伍，若某一轮队伍部位偶数，则淘汰向下取整支队伍，某一轮未进行比赛的队伍为轮空
    4. 奇数个队伍打比赛，一般从轮空队伍的场次入手

# 天平测假币问题

  1. 关键词：有一枚假币重量不同，则使用一个天平，最优情况下最多需要多少次对比称重
  2. 思路：
    1. 每次将所有硬币三等分，若不能三等分则不同组的数量差最多为一
    2. 每一次称重，若两组相同，则假币在未称重的一组，否则则在质量有问题的一组。若某一次称重无法平分，则称重两组数量相同的
  3. 结论：若有n个硬币
    1. 最优情况下最多需要x称重，则满足$3^n\geq n$
    2. 若已知假币比真币轻还是重，则x次可判断$3^x$个硬币
    3. 若未知假币与真币的重量关系，则需要x+1次才可判断$3^x$，即多一次判断轻还是重
