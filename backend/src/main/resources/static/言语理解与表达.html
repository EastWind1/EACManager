# 片段阅读

# 主旨概括

  1. 题目类型关键词：主旨、概括、旨在说明、主要
  2. 寻找关联词或行文脉络
    1. 关联词
      1. 递进： 不但、不仅、除了，还，强调递进后
      2. 转折：虽然、尽管、事实上、实际上，强调转折后
      3. 因果：因为、因此、导致、造成，强调结果，忽略原因
      4. 条件：必要条件、充分条件、假设条件，强调条件
      5. 并列：又、及、此外，全面概括
    2. 行文脉络 （总：核心观点、解决对策，分：举例、背景、原因、引用、反面论证）
      1. 总-分：提出观点-论证，提出问题-分析
      2. 分-总：列出现象原因-结论，提出问题-解决，解释说明-观点
      3. 总-分-总：提出观点-解释-重申，提出问题-分析-解决
      4. 分-总-分：现状/背景/援引/举例-提出观点/对策-举例/解释原因/反面论证
      5. 分-分：并列
  3. 确定关键部分
  4. 同义替换、压缩、全面概括

# 意图判断

  1. 判断题目类型的关键字：意在、想、想要、意图、推出
  2. 寻找关键词/行文脉络
  3. 区分类型：启示、问题、观点
  4. 优先合理引申，其次找出重点概括
  5. 常见混淆选项
    1. 在分析意图的题型中的主旨项
    2. 过度引申项
    3. 非重点项
    4. 无中生有项

# 细节理解

# 细节判断

  1. 关键词：正确/不正确，符合/不符合，推出/推不出
  2. 思路：明确问题-速读选项并在文中对应-排除干扰项-确定答案
  3. 正确选项特点：与原文一一对应、对原文的理解、从原文合理推出
  4. 干扰选项特点：无中生有，偷换概念、数量、逻辑、时态、语气

# 细节查找

  1. 关键词：原因、目的、区别、特点、未提及
  2. 思路：明确问题-定位原文-匹配答案

# 标题选择

  1. 关键词：题目、标题
  2. 思路：概括主旨、参考文体
  3. 文体分类
    1. 新闻类：提炼导语
    2. 说明文：说明对象+说明要素
    3. 议论文：提炼论点或对策
    4. 其他：散文（中心主旨或感情倾向）、故事（结果入手、分析原因与道理）、游记（行为及地点）等

# 态度理解

  1. 关键词：态度、观点
  2. 分类
    1. 有观点
      1. 明确观点：我认为、我觉得、笔者认为、xxx认为
      2. 隐含观点：关联词语（但是、所以）、感情倾向（积极、消极）、标点符号（？反问，”“反讽）
    2. 无观点：选择选项中作者未评价、未表达态度的选项

# 词语指代

  1. 关键词：上文中的xx指的是、对xx理解
  2. 词语理解
    1. 标点符号：冒号、破折号表示解释说明
    2. 关联词语：转折、递进、因果、并列
    3. 关键引导词：是、即、所谓、也就是说（解释型，可同义替换），比如、例如、因为、由于、原因是（举例或原因型，需概括归纳）
  3. 代词指代：就近查找，若没有则与文段话题一致

# 逻辑填空

# 步骤一：语境分析

  1. 对应词句类
    1. 关键字：可谓、无异于
    2. 对应：前后话题连贯性、指示代词、 形象化表达
  2. 关联词语类
    1. 递进：语义一致，且后者比前者语义略重，感情倾向一致
    2. 转折：语义相反，感情色彩相反
    3. 并列
      1. 平行并列：前后语义相近，感情倾向一致或相反
      2. 对照并列：前后语义相反、反向照应
    4. 因果：前后因果呈相关性，感情色彩一致
  3. 标点符号类
    1. 分号、顿号、逗号：表示并列
    2. 冒号、破折号：表示解释说明
    3. 双引号：讽刺、强调、引用、形象化表达

# 步骤二：词语辨析

  1. 语义轻重
    1. 区分词语程度轻重、语气深浅
    2. 选择与文段表达含义轻重一致的词语
  2. 感情色彩
    1. 词性划分：褒义词、贬义词、中性词
    2. 所填词语的感情色彩与文段的感情色彩保持一致
  3. 词语搭配：
    1. 语法搭配：主谓搭配、动宾搭配、修饰语与中心语的搭配
    2. 寻找搭配对象
    3. 固定搭配：
      1. 约定俗成的搭配：常见成语
      2. 时政热点或讲话：赓(geng)续血脉，打通梗阻，经济新常态，xxx协同发展
  4. 语素差异：常用语多个含义相近的词语辨析
    1. 语素：最小有音又有义的语言单位
    2. 方法：搁置相同，利用拆字组词法分析
    3. 区分词与词之间的语素差异，结合语境

# 语句表达

# 语句填空

  1. 关键字：填入横线处最恰当的一项
  2. 步骤
    1. 首选形式
      1. 关联词语搭配
      2. 句式一致
    2. 兼顾内容
      1. 话题一致
      2. 符合行文脉络：总分总、总分、分总
      3. 逻辑关系：转折、因果

# 下文判断

  1. 关键词：接下来最可能/不可能讲
  2. 思路：确定落脚话题-推断下文-匹配选项
  3. 推断下文方法
    1. 话题一致
    2. 行文脉络
      1. 提出观点-论证/解释观点
      2. 提出问题-分析/解决问题
    3. 感情色彩一致

# 语句排序

  1. 确定首句
    1. 排除项
      1. 补充类描述
      2. 反面论证
      3. 指代类描述
      4. 关联词后半部分
      5. 结论类描述
    2. 适合项
      1. 引出话题
      2. 定义表述
      3. 较大的话题概念范围：概念范围大者比小者适合做首句
  2. 验证连贯性
    1. 话题一致
    2. 关联词语
    3. 代词指代
    4. 视同顺序
    5. 行文脉络
    6. 事理逻辑

# 篇章阅读

  1. 解题思路
    1. 阅读题设，审请设问
    2. 浏览文章
    3. 匹配选项
  2. 中心主旨类、细节理解类同片段阅读
  3. 话题衔接类
    1. 关键词：放在文中哪个位置合适
    2. 某段内的话题衔接：首选形式一致（关联词、句式），兼顾内容（话题，行文脉络、逻辑）
    3. 段落间的话题衔接：话题一致
