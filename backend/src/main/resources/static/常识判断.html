# 政治

# 马克思主义哲学原理

  1. 哲学基本问题：物质与意识何为本源、思维与存在的关系
  2. 唯物主义
    1. 古代朴素唯物主义：一种或某几种具体的物质形态是世界的本源（如五行）
    2. 近代形而上学唯物主义：用孤立的、精致的、偏面的观点看世界（如认为原子是世界本源）
    3. 辩证唯物主义：用联系、发展、全面的观点看问题
  3. 唯心主义
    1. 主观唯心主义：把人的主观精神看成是世界的本源
    2. 客观唯心主义：把”客观“精神当成是世界的本源（如神）
  4. 唯物论
    1. 物质：物质不依赖于人的感觉而存在，能被意识反应，唯一特性是客观实在性
    2. 物质的根本属性是运动，运动是绝对的，静止是相对的
    3. 意识：是人脑特有的机能，是对客观存在的反应
    4. 世界观：物质决定意识，意识反作用于物质
    5. 方法论：一切从实际出发，重视发挥主观能动性
  5. 辩证法
    1. 联系：普遍性（没有孤立事物）、客观性（本身固有的，不以人的意志为转移）、多样性（不同事物，同一事物不同条件都不同）
    2. 对立统一：
      1. 原理：矛盾对立与统一。矛盾具有普遍性和特殊性
      2. 方法论：一分为二的观点，具体问题具体分析，坚持两点论和重点论
      3. 内因外因：内因是事物的内部矛盾，外因是事物的外部矛盾。内因和外因的关系是：内因是事物发展变化的根据，它规定了事物发展的基本趋势和方向；外因是事物发展变化的不可缺少的条件，有时外因甚至对事物的发展起着重大的作用；外因的作用无论多大，也必须通过内因才能起作用。
    3. 量变质变
      1. 原理：量变是质变的必要准备，质变是量变的必然结果
      2. 方法论：坚持循序渐进与变革相结合，坚持适度原则，重视量的积累
    4. 否定之否定
      1. 原理：事物由肯定到否定，再到否定之否定的发展过程，事物发展的道路是前进性和曲折性的统一
      2. 方法论：科学分析，同时看到肯定和否定，把握事物发展的总趋势，正确对待前进和曲折
  6. 认识论
    1. 认识
      1. 感性认识：感觉、直觉、表象，直接性、具体性，是低级形式，反映事物的现象
      2. 理性认识：概念、判断、推理，间接性、抽象性，是高级形式，反映事物的本质
    2. 实践：时间的观点是辩证唯物主义认识论的首要的基本的观点
      1. 实践是认识的基础，是认识的来源，是认识发展的动力，是认识的最终目的，是检验真理的唯一标准
      2. 认识对实践具有能动的反作用。正确的认识促进实践，错误的认识阻碍或破坏实践
  7. 历史观
    1. 唯心史观：把社会历史看成精神发展史（神学史观、英雄史观）
    2. 唯物史观：认为社会存在决定社会意识。即强调英雄对历史的推动作用，又强调人民群众对历史发展的决定性作用
    3. 社会存在：指社会生活的物质方面，包括地理环境、人口因素、生产方式
    4. 社会意识：指社会生活的精神方面，是社会存在的总体反映。包括政治思想、法律思想、哲学、哲学、艺术、宗教、风俗习惯、社会心理
    5. 社会存在决定社会意识，社会意识有相对独立性，社会意识反作用于社会存在
    6. 上层建筑：指建立在一定经济基础上的社会意识形态以及与之相适应的政治法律制度和设施等的总和，包括政治上层建筑和思想上层建筑
    7. 人民群众的作用
      1. 人民群众是社会历史的主体，是社会变革的决定力量
      2. 人民群众是生产力的体现者
      3. 人民群众的社会实践是社会物质财富和精神财富的源泉
      4. 人民群众的社会实践是社会变革的决定力量
      5. 人民群众的历史创造作用受到一定的社会历史条件的制约

# 党史

  1. 1921年，中共一大：建立中国共产党，确立党的奋斗目标：推翻资产阶级，建立无产阶级专政，废除私有制，直至消灭阶级差别。选举陈独秀为中央局书记。通过了党的第一部纲领《中国共产党纲领》
  2. 中共二大：第一次明确提出了彻底的反帝反封建的民主革命纲领。通过中国共产党章程，提出统一战线政策
  3. 北伐：打击受帝国主义支持的北洋军阀，主要有吴佩孚、孙传芳、张作霖
  4. 四一二反革命政变：北伐战争转折点，标志第一次国共合作失败
  5. 南昌起义：打响了武装反抗国民党反动派的第一枪，标志着党独立领导革命战争、创建人民军队和武装夺取政权的开端
  6. 八七会议：毛泽东提出了”枪杆子里出政权“
  7. 秋收起义：毛泽东在湘赣边界发动
  8. 三湾改编：建设新型人民军队最早的一次成功探索和实践，最早从组织上确立了党领导军队的原则
  9. 古田会议：确立党对军队的绝对领导，提出建设新型军队，提出”思想上建党、政治上建军“的原则
  10. 长征：瑞金→突破敌四道防线→强渡乌江→占领遵义→四渡赤水→巧渡金沙江→强渡大渡河→飞夺泸定桥→翻雪山→过草地→陕北吴起会师→甘肃会宁会师→宁夏西吉县将台堡会师
  11. 遵义会议：第一次独立自主运用马克思主义基本原理解决中国革命问题，党从幼年走向成熟的标志。形成以毛泽东为核心的第一代中央领导集体
  12. 瓦尔堡会议：提出建立广泛的抗日民族统一战线
  13. 卢沟桥事变
  14. 洛川会议：提出抗日救国十大纲领，全面抗战路线正式形成
  15. 中共七大：确立毛泽东思想为党的领导思想并写入党章。毛泽东《论联合政府》提出三大作风
  16. 中共七届二中全会（西柏坡会议）：党的工作重心由农村转移至城市，两个务必（谦虚谨慎戒骄戒躁，艰苦奋斗）
  17. 1949年，新中国成立：标志着中国进入了社会主义革命和建设时期
  18. 抗美援朝
  19. 土地改革
  20. 第一届全国人大：第一部宪法颁布
  21. 三大改造完成：标志着社会主义制度在我国基本建立
  22. 十一届三中全会
    1. 重新确立了解放思想、实事求是的思想路线

# 中国特色社会主义理论体系

  1. 毛泽东思想
    1. 进党章：中共七大
    2. 精髓：实事求是
    3. 回答：中国的民主革命、新民主主义革命等中国革命的道路问题
  2. 邓小平理论
    1. 进党章：中共十五大
    2. 精髓：解放思想，实事求是
    3. 回答：什么是社会主义，怎样发展社会主义
  3. 三个代表重要思想
    1. 进党章：中共十六大
    2. 精髓：解放思想，实事求是，与时俱进
    3. 回答：建设什么样的党，怎么建设党
  4. 科学发展观
    1. 进党章：中共十七大，作为指导思想为中共十八大
    2. 精髓：解放思想，实事求是，与时俱进，求真务实
    3. 回答：实现什么样的发展，怎么发展
  5. 新时代中国特色社会主义思想
    1. 进党章：中共十九大
    2. 精髓：实事求是，人民至上，知行合一
    3. 回答：系统回答了新时代要发展什么样的中国特色社会主义，怎么样坚持和发展中国特色社会主义
    4. 二十大把党的十九大以来习近平新时代中国特色社会主义思想新发展写入党章，新增两个明确
      1. 明确必须坚持和完善社会主义基本经济制度，使市场在资源配置中起决定性作用，更好发挥政府作用，把握新发展阶段，贯彻创新、协调、绿色、开放、共享的新发展理念，加快构建以国内大循环为主体、国内国际双循环相互促进的新发展格局，推动高质量发展，统筹发展和安全，
      2. 明确全面从严治党的战略方针，提出新时代党的建设总要求，全面推进党的政治建设、思想建设、组织建设、作风建设、纪律建设，把制度建设贯穿其中，深入推进反腐败斗争，落实管党治党政治责任，以伟大自我革命引领伟大社会革命

# 公务员相关名词

  1. 公务员录用，是指国家机关按照规定的条件和程序，面向社会采用公开考试、严格考察的办法选拔公务员的活动。
  2. 调任就是将公务员队伍以外的其他从事公务的人员调入机关担任一定职务。挂职的公务员还属于选派单位的人，占用原单位的编制，锻炼期满后，仍回原单位工作或者留用、提拔。
  3. 转任指公务员因工作需要或者其他原因在机关系统内跨地区、跨部门的调动，或者在同一部门内的不同职位之间进行的转换任职。转任是公务员交流的一种重要形式，公务员在机关系统内部转换职位任职，只涉及职务关系的变更，不涉及公务员身份的变更。
  4. 挂职干部是指选派公务员到下级机关、上级机关、其他地区机关以及国有企业事业单位担任相应职务进行锻炼的交流方式。挂职的公务员还属于选派单位的人，占用原单位的编制，锻炼期满后，仍回原单位工作或者留用、提拔。
  5. “老虎”喻指位居高层的腐败官员，“苍蝇”则指身处基层的腐败官员。”狐狸“指潜逃海外的职务犯罪嫌疑人
  6. 命令（令）适用于公布行政法规和规章、宣布施行重大强制性措施、批准授予和晋升衔级、嘉奖有关单位和人员。
  7. 通报适用于表彰先进、批评错误、传达重要精神和告知重要情况
  8. 公告适用于向国内外宣布重要事项或者法定事项
  9. 通知适用于发布、传达要求下级机关执行和有关单位周知或者执行的事项,批转、转发公文
  10. 行政处分分为：警告、记过、记大过、降级、撤职、开除

# 经济

# 经济学名词

  1. 商品：价值和使用价值。使用价值是价值的基础，价值是使用价值的表现形式
  2. 货币：从商品中分离出来的、固定地承担一般等价物的特殊商品。价值尺度、流通手段、贮藏手段、支付手段、世界货币
  3. 经济学名词：
    1. GDP：国内生产总值，为生产活动的最终成果，是一个地域概念（以地域为界，即国外收入无法计入GDP）
    2. GNP：国民生产总值，为收入初次分配的最终结果，是一个国民概念（以国籍为限）
    3. 基尼系数：衡量居民收入差距，最大为1，越接近于0越平等。<0.2过于平均，0.2-0.3较为平均，0.3-0.4比较合理，0.4-0.5差距过大，>0.5差距悬殊
    4. 恩格尔系数：食品支出总额站个人消费支出总额的比重，反映一个家庭或国家的富裕程度。越大越贫困。>59%贫困，50-59温饱，40-50小康，30-40富裕，<30最富裕
    5. 边际成本：增加以单位产量而产生的成本增加量。随着产量增加，边际本先减少后增加
    6. 羊群效应：从众跟风心里
    7. 口红效应：经济不景气的情况下，为弥补消费欲望，会购买一些廉价的非必要之物
    8. 棘轮效应：消费习惯形成之后具有不可逆性
    9. 鳄鱼效应：越挣扎，咬住的越多。要及时止损。
    10. 寡头垄断：所谓寡头，是指少数的卖者面对众多的买者
    11. 垄断企业：如果一个企业是其产品唯一的卖者,而且其产品没有相近的替代品,那么这个企业就是一个垄断企业
    12. 囚徒困境：两个被捕的囚徒之间的一种特殊博弈，说明即使合作对双方都有利时，保持合作也是困难的。反映个人最佳选择并非团体最佳选择
  4. 贷款相关
    1. 等额本金：每月还款固定本金数加剩余本金数的月利息--每月月供额=(贷款本金÷还款月数)+(贷款本金-已归还本金累计额)×月利率
    2. 等额本息：将按揭贷款的本金总额与利息总额相加，然后平均分摊到还款期限的每个月中，利息计算要依据每个月的剩余本金进行计算。$每月还款数=贷款总额\cdot\frac{月利率(1+月利率)^{总还款月数}}{(1+月利率)^{总还款月数}-1}$ $$ \begin{align} &设贷款金额为Q,贷款月份为n,月利率为R\ &设每个月还款金额为x\ &则第一个月剩余贷款金额为Q_1=Q(1+R)-x\ &第二月剩余贷款金额为Q_2=Q_1(1+R)-x=Q(1+R)^2-x[1+(1+R)]\ &第三月Q_3=Q_2(1+R)-x=Q(1+R)^3-x[1+(1+R)+(1+R)^2]\ &第k月Q_k=Q(1+R)^k-x[1+(1+R)+(1+R)^2+...+(1+R)^k]\ &第k月还清，即Q_k=0，得出x公式\ &x=Q\cdot\frac{R(1+R)^n}{(1+R)^n-1} \end{align} $$
  5. 一五计划：1953-1957
  6. 十四五计划：2021-2025

# 宏观调控

  1. 包括：财政政策、货币政策、收入分配政策、消费政策
  2. 目标：促进经济增长、增加就业、稳定物价、保持国际收支平衡
  3. 通货膨胀
    1. 表现：货币贬值、物价上涨，购买力降低
    2. 成因：社会总需求大于社会总供给，货币发行量超过了流通中实际需要的货币量
    3. 影响：一定时期内适度的通货膨胀有利于促进消费，扩大内需，促进经济发展。否则则货币贬值，购买力降低，不利于经济发展
    4. 对策：发展生产，增加有效补给，同时采取控制货币供应量，事事从紧的货币政策（如提高利率）和量入为出的财政政策（如提高税率）等措施
  4. 通货紧缩：
    1. 表现：生产下降、市场萎缩、企业利润率降低，生产投资减少，失业增加、收入下降，经济增长乏力。物价低迷，大多数商品和劳务价格下跌
    2. 成因：社会总需求小于社会总供给，长期的产业结构不合理，是由买方市场及出口困难导致的
    3. 影响：短期物价下跌对居民生活有利。长远来看会严重影响投资者的信心和居民的消费心理，导致恶性的价格竞争，对经济的长远发展和人民的长远利益不利
    4. 对策：优化产业结构，综合运用投资、消费、出口等措施拉动经济增长，实施积极的财政政策、稳健的货币政策、正确的消费政策、坚持扩大内需的方针
  5. 财政政策：
    1. 含义：政府通过对财政收入和支出总量的调节来影响总需求，是指与总供给相适应的经济政策
    2. 实施主体：国家财政机关
    3. 政策手段：税收、预算支出、公债、财政补贴、财政贴息（银行贷款的利息一部分由政府承担）等
  6. 货币政策
    1. 含义：指央行为实现一定的宏观经济目标而对货币的供应领和信贷量进行调节和控制所采取的指导方针及其相应的政策措施
    2. 实施主体：中央银行
    3. 政策手段：公开市场业务，再贴现率（银行由于资金问题无法承兑票据，向央行贴现的贴息利率）、法定存款准备金（金融机构为保证客户提取存款和资金清算而缴存在中央银行的存款）、利率等
  7. 初次分配：国民总收入首先在生产部门进行分配，包括国家收入、企业收入、个人收入（工资收入）
  8. 再分配：在初次分配结果的基础上，政府对要素收入进行再次调节。再分配主要有政府调控机制起作用，是保持社会稳定、维护社会公正的基本机制。包括社会保障（社会保险、社会福利、社会优抚、社会救济）、政府转移支付等
  9. 社会保险：对参加劳动关系的劳动者在丧失劳动能力或失业是给予必要的物质帮助的保障制度，社会保障制度的核心内容。包括养老、失业、医疗、工伤、生育等
  10. 社会福利：包括公共福利、社会福利和集体福利
  11. 社会救济：属于最低层次的社会保障措施
  12. 社会优抚与安置：对象包括现役军人、军属、退伍、复员、转业、残疾军人、军烈属等特殊社会成员
  13. 市场机制：供求机制、价格机制（核心）、竞争机制
  14. 垄断竞争类型：完全竞争（如农产品）、垄断竞争（如轻工产品）、寡头垄断（如石油、通信）、完全垄断（公共事业，如水、电）

# 民法

  1. 概念：调整平等主体之间的财产关系和人身关系的法律范围的总成
  2. 对象：平等主体的自然人、法人和非法人组织
  3. 自然人：
    1. 无民事行为能力人：不满8周岁，以及不能辨认自己行为的人
    2. 限制民事行为能力的人：已满8周岁不满18周岁。可独立实施纯获利益的民事法律行为或与其年龄、智力、精神健康状况相适应的民事法律行为
    3. 完全民事行为能力的人
      1. 已满18周岁且精神正常的人
      2. 视为完全民事行为能力的人：年满16周岁，以自己的劳动收入为主要生活来源
  4. 法人：具有民事权利能力和民事行为能力，依法独立享有民事权利和承担民事义务的组织
  5. 民事法律行为的效力：限制民事行为能力人超越其能力的行为、无权代理的行为为效力待定
  6. 物权
    1. 所有权：所有人对所有的拆产享有的占有、使用、收益、处分的权利
    2. 用益物权：以标的物的使用和收益为目的而设立的定限物权，如土地承包经营权、建设用地使用权、宅基地使用权、居住权、地役权
    3. 担保物权：为担保债权的实现而设立的定限物权，如抵押权、质权、留置权
    4. 宅基地只有使用权，所有权归集体。城市户籍子女可继承。
  7. 人格权
    1. 是所有的民事权利中最重要的
    2. 生命权、身体权、健康权、姓名权、肖像权、名誉权、荣誉权、隐私权、婚姻自主权等
    3. 冒充他人身份，顶替他人取得高等学历教育入学资格、公务员录用资格、就业安置待遇的，处三年以下有期徒刑、拘役或管制，并处罚金
  8. 隐私权：个人敏感信息包括：生物识别、宗教信仰、特定身份、医疗健康、金融账户、行踪轨迹等信息，以及不满十四周岁未成年人的个人信息
  9. 继承权
    1. 法定继承
      1. 第一顺序：配偶、子女、父母
      2. 第二顺序：兄弟姐妹、祖父母、外祖父母
      3. 丧偶儿媳对公婆，丧偶女婿对岳父母，进了主要赡养义务的作为第一顺序继承人
    2. 遗嘱继承
      1. 自书遗嘱
      2. 代书遗嘱、打印遗嘱、录音录像遗嘱、口头遗嘱：应当有两个以上见证人在场见证。
      3. 公正遗嘱
    3. 遗赠
      1. 对象：受赠人必须是法定继承人以外的自然人或国家及其他社会组织
      2. 受遗赠人应当在知道受遗赠后六十日内，作出接受或拒绝的表示，否则视为放弃
  10. 收养条件
    1. 无子女或只有一名子女
    2. 有能力抚养
    3. 未患有医学上认为不应当收养子女的疾病
    4. 无不利于被收养人健康成长的违法犯罪纪律
    5. 年满三十周岁
    6. 无配偶者收养异性子女的年龄相差四十周岁以上，但三代以内旁系同辈学琴子女除外
    7. 无子女的收养人可收养两名子女

# 刑法

  1. 空间效力
    1. 境内犯罪：属地管辖原则
      1. 在领域内犯罪的，除法律有特别规定的以外，都是用我国刑法
      2. 在我国船泊或者航空器内犯罪的，也适用我国刑法
      3. 犯罪行为或犯罪结果有一项发生在我国领域内的，就认为实在我国领域内犯罪
      4. 例外：享有外交特权或豁免权的外国人
    2. 境外犯罪：
      1. 属人管辖原则
        1. 我国公民在我国领域外实施犯罪的，原则上适用我国刑法
        2. 除国家工作人员和军人以外的其他人轻微犯罪（刑法规定的三年以下有期徒刑），可以不予追究
      2. 保护管辖：外国人在境外危害中国国家和公民利益的犯罪，最低刑为三年以上，且犯罪地法律也认为应当受处罚的行为，可以适用我国法律
      3. 普遍管辖：我国缔结或参加的国际条约所规定的罪行，我国在所承担条约义务的范围内行使行使管辖权的，适用我国刑法
  2. 刑事责任
    1. 完全无责任年龄：12岁
    2. 相对责任年龄：
      1. 已满12周岁不满14周岁的人。犯故意杀人、故意伤害罪，致人死亡或者以特别残忍手段致人重伤造成严重残疾，情节恶劣，经最高人民检察院核准追诉的，应当负有刑事责任
      2. 已满14不满16：犯故意杀人、故意伤害致人重伤或者死亡、强奸、抢劫、贩卖毒品、放火、爆炸、投放危险物质的，应当负刑事责任
    3. 完全责任年龄：16周岁以上
    4. 减轻责任年龄：已满75周岁以上的人故意犯罪的，可以从轻或减轻处罚；过失犯罪的，应当从轻或减轻处罚
    5. 精神病人在不能辨认或不能控制自己行为的时候，经法定程序鉴定确认的，不负刑事责任
    6. 醉酒的人犯罪，应当负刑事责任
    7. 又聋又哑的人或盲人犯罪，可以从轻、减轻或者免除处罚
  3. 正当防卫与紧急避险
    1. 正当防卫未超过必要限度造成不应有的损害的，不负刑事责任。超过的应当减轻或免除处罚
    2. 紧急避险：保护的利益大于损害的利益的，不负刑事责任，但不适用于职务上、业务上负有特定责任的人
  4. 刑罚
    1. 死刑：犯罪时不满18周岁的人，审判时怀孕的妇女，审判时已满75周岁的人不适用。以特别残忍手段致人死亡的除外
    2. 死缓为两年
    3. 死缓期满后的处理
      1. 没有故意犯罪的减为无期
      2. 由重大立功表现，减为25年有期
      3. 故意犯罪，情节恶劣的，核准后执行
      4. 故意犯罪未执行死刑的，死缓重新计算
    4. 附加刑：罚金、没收财产、剥夺政治权利、驱逐出境
    5. 剥夺的政治权利包括
      1. 选举权和被选举权
      2. 言论、出版、集会、截设、游行示威自由的权利
      3. 担任国检机关职务的权利
      4. 担任国有公司、企业、事业单位和人民团体领导职务的权利
    6. 剥夺政治权利的刑期从徒刑、拘役执行完毕之日或假释之日起计算
  5. 受贿罪：国家工作人员利用职务便利，索取他人财务，或非法收受他人财务，为他人谋取利益的
  6. 利用影响力受贿罪：国家工作人员的近亲属或者其他与该国家工作人员关系密切的人，通过该国家工作人员职务上的行为，为请托人谋取不正当利益，索取或收受请托人财务，数额较大或者有其他较重情节的
  7. 贪污罪：是指国家工作人员利用职务上的便利，侵吞、窃取、骗取或者以其他手段非法占有公共财物的行为。
  8. 挪用公款罪：国家工作人员利用职务上的便利，实施挪用公款归个人使用，进行非法活动，或者挪用公款数额较大、进行营利活动，或者挪用公款数额较大、超过三个月未还，从而构成的犯罪。
  9. 私分国有资产罪：国家机关、国有公司、企业、事业单位、人民团体，违反国家规定，以单位名义将国有资产集体私分给个人，数额较大的行为。
  10. 滥用职权罪：国家机关工作人员超越职权，违法决定、处理其无权决定、处理的事项，或者违反规定处理公务，致使公共财产、国家和人民利益遭受重大损失的行为

# 行政法

  1. 行政主体：能够以自己的名义实施国家行政管理职能并承受相应法律后果的国家行政机关或社会组织
  2. 行政行为
    1. 抽象行政行为：行政主体不针对特定的行政相对人实施行政管理的行为
    2. 具体行政行为：针对特定对象
  3. 行政处罚规则
    1. 不满14周岁的不予处罚
    2. 已满14不满18周岁的从轻或减轻处罚
    3. 违法行为发生之日起或连续、继续行为终了之日起两年内违背发现的，不再处罚
    4. 同一个违法行为，不得给予两次以上罚款的行政处罚。同一个违法行为违反多个法律规范应当给予罚款处罚的，按罚款数最高的规定处罚
  4. 行政处罚程序
    1. 简易程序：违法事实确凿并有法定依据。对公民处200元以下、对单位处3000元以下罚款或警告。处罚判决书必须当场交付当事人
    2. 一般程序：除建议程序以外的
    3. 当场收缴罚款的情况：一百元以下罚款的或不当场收缴事后难以执行的

# 中国近代史

  1. 鸦片战争（英国）
    1. 导火索：虎门销烟
    2. 林则徐：近代开眼看世界第一人，苟利国家生死以，岂因祸福避趋之
    3. 魏源：《海国图志》，中国近代史上第一部较为详尽较为系统的世界历史、地理著作
    4. 不平等条约：《南京条约》，中国近代第一个不平等条约，标志进入了旧民主主义革命时期
  2. 太平天国运动
    1. 金田起义
    2. 《天朝田亩制度》，《资政新篇》
  3. 第二次鸦片战争（英、法、美、俄）
    1. 《天津条约》《北京条约》《瑷珲条约》
    2. 火烧圆明园
  4. 洋务运动
    1. 师夷长技以制夷
    2. 中学为体、西学为用
    3. 自强、求富
    4. 甲午中日战争北洋海军全军覆没，签订《马关条约》，标志洋务运动失败
    5. 使中国出现了第一批近代企业
  5. 戊戌变法
    1. 又称百日维新、维新变法，是政治改良运动
    2. 戊戌六君子
    3. 建立京师大学堂（北京大学）
  6. 义和团与八国联军侵华
    1. 由于宗教问题，发源于山东，口号扶清灭洋
    2. 义和团运动为八国联军侵华的导火索
    3. 《辛丑条约》，标志完全沦为半殖民地半封建社会
  7. 辛亥革命
    1. 武昌起义-中华民国成立-清帝退位
    2. 兴中会：第一个资产阶级革命团体
    3. 意义：是近代中国比较完全意义上的资产阶级民主革命，推翻了清王朝统治，结束了2000多年的君主专制制度
  8. 新文化运动
    1. 主要阵地：《新青年》，北京大学
    2. 代表人物：陈独秀、李大钊、鲁迅、护士
    3. 思想：民主、科学
    4. 李大钊发表《布尔什维克主义的胜利》《庶民的胜利》
    5. 为马克思主义在中国的传播和五四爱国运动的爆发奠定了思想基础
  9. 五四运动
    1. 导火索：巴黎和会上中国外交的失败
    2. 时间：1919年5月4日
    3. 标志着新民主主义革命的开端
    4. 旧民主主义革命：由资产阶级领导的，以建立资本主义社会和资产阶级专政的国家为目的、反对外国侵略和本国封建统治的革命
    5. 新民主主义革命：由无产阶级领导的，人们大众的，反对帝国主义、封建主义和官僚资本主义的革命，是新是的特殊的资产阶级民主革命

# 基本国情

  1. 地势：西高东低，呈三级阶梯状分布
  2. 四大高原：青藏高原、内蒙古高原、黄土高原、云贵高原
  3. 四大盆地：塔里木盆地（世界第一大内陆盆地，世界第二大沙漠：塔克拉玛干沙漠），准噶尔盆地，柴达木盆地（世界地势最高盆地），四川盆地
  4. 三大平原：东北平原、华北平原、长江中下游平原
  5. 雅丹地貌：典型风蚀地貌，风蚀土墩和风蚀凹地
  6. 丹霞地貌：红层、陡崖坡
  7. 喀斯特地貌：石灰岩地貌
  8. 长江：世界第三大河，流经11个省级行政区
  9. 黄河：流经9个省级行政区
  10. 秦岭淮河一线：为一月份零度等温线和800毫米等降水量线

# 自然地理

  1. 大气层
    1. 对流层：天气现象，温度随高度增加而降低
    2. 平流层：飞行器，温度随高度增加而升高
    3. 中间层：垂直对流强烈，温度随高度增加而降低
    4. 臭氧层：平流层一部分
    5. 电离层：反射和传送高频无线电信号，实现远距离无线电通信
  2. 一天文单位（1AU）：1.5亿千米
  3. 月相：新月（朔月）（初一）、蛾眉月（盈月）（初二至初七）、上弦月（初八）、上凸月（初九至十四）、望月（满月，十五或十六），下凸月（十六至二十三），下弦月（二十三），蛾眉月（残月，二十五，二十六）
  4. 地壳元素：含量最多：氧。金属元素最多：铝
